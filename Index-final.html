About This Dashboard

What is this? The Global Phishing Tracker is an educational cybersecurity dashboard that visualizes how phishing emails are detected across the world in real-time.

Data Source: This dashboard simulates real threat intelligence data. In production, it aggregates data from:

  • Corporate email security gateways (Microsoft Defender, Proofpoint, Mimecast)
  • User phishing reports and spam complaints
  • Email authentication protocols (DMARC, SPF, DKIM)
  • URL reputation and domain analysis services
  • Machine learning models detecting suspicious patterns

Live Updates: This dashboard updates every 5 seconds with new threat data. The detection feed, metrics, and trends are all refreshed in real-time.

Educational Purpose: This project demonstrates cybersecurity concepts and threat visualization techniques. Perfect for learning about phishing, security operations, and data visualization.
